# Net.Simulation

**ネットワークの動きを、スマートフォンで体感する学習・検証ツール**

ネットワークの仕組みを「なんとなく知っている」から「実際に動かして理解できる」レベルへ。  
CCNAを目指している方、インフラ業務を始めた方、現場での説明ツールを探している方に向けて作りました。

---

## こんな方に向けて作りました

| 対象 | 使い方のイメージ |
|------|----------------|
| ネットワーク初学者 | シナリオ学習でルーティングやTCPの仕組みをステップごとに理解する |
| CCNA・CCNP勉強中の方 | 障害シミュレーションで試験問題の「なぜ」を体感する |
| インフラエンジニア | 障害発生時の迂回経路や影響範囲を素早く図示・検証する |
| SE・プリセールス | 顧客への説明資料として、その場でデモを見せる |

---

## 主な機能

### ネットワークトポロジの設計

デバイスをドラッグして配置し、タップで接続するだけでネットワーク図が作れます。  
PC・サーバー・ルーター・スイッチ・ファイアウォール・クラウドノードなど21種類のデバイスに対応しています。

### パケットアニメーション

再生ボタンを押すとパケットがリンク上をホップバイホップで流れます。  
正常転送は青、ノード処理中はオレンジ、到達成功は緑で色分けされるため、どのパスを通っているか一目で確認できます。

### 障害シミュレーション

シミュレーション中にリンクやデバイスを長押しすると、その場で状態を変更できます。

| 操作 | 効果 |
|------|------|
| リンクを長押し → Link Down | 赤い破線になり、パケットが自動的に迂回経路を探す |
| リンクを長押し → Congested | 輻輳状態になり、オレンジ色で表示される |
| デバイスを長押し → Crash | クラッシュ状態になり、経路から除外される |
| デバイスを長押し → Reboot | 指定秒数後に自動復旧する（10秒・30秒） |

障害発生時の操作はSyslogに自動記録されます。

### CLI

デバイスをダブルタップするとCLIタブが開きます。実際のコマンドと同じ形式で操作できます。

```
ping 192.168.1.1
ping Router-1          # デバイス名でも可
traceroute PC-2
ifconfig
show ip route
arp
nslookup Router-1
```

### シナリオ学習

AppBarの学校アイコンから7本のシナリオを選んで学習できます。  
各ステップに「やること」と「なぜそうなるか」の解説が付いています。

| シナリオ | 難易度 |
|---------|--------|
| 基本的なLAN構成を作る | 初級 |
| ルーターで2つのネットワークを繋ぐ | 初級 |
| TCPの3ウェイハンドシェイク | 初級 |
| リンク障害と迂回経路 | 中級 |
| VLANで部署ネットワークを分離する | 中級 |
| ファイアウォールで通信を制御する | 中級 |
| OSPFの経路収束を観察する | 上級 |

### プロトコル可視化

TCP・OSPF・VLAN・DHCP・ARPの動作をステップバイステップのアニメーションで確認できます。  
「DHCPのDORAって何?」「OSPFのネイバーシップがどう確立されるか」を視覚的に理解できます。

### セキュリティテスト（仮想環境内のみ）

SYNフラッド・ポートスキャン・ARPスプーフィングなどの攻撃シミュレーションを仮想ネットワーク内で実行できます。  
IDS/IPSによる検知・遮断の様子と統計をリアルタイムで確認できます。

> 本機能はすべて仮想ネットワーク内に限定されており、実際のネットワークへの影響は一切ありません。教育目的専用です。

### デモ構成

トポロジ一覧からサンプル構成を読み込んですぐ動かせます。

| 構成名 | 概要 |
|--------|------|
| インターネット接続構成 | PC・ルーター・インターネットの基本構成 |
| 社内ローカル構成 | スイッチ・サーバー・ルーターで組む社内ネットワーク |
| セキュリティ・FW/IDS構成 | ファイアウォールとIDS/IPSを含む構成 |
| WAN広域構成（MPLS） | 拠点間をMPLSとVPNで繋ぐ構成 |
| 全デバイス構成 | 21種類すべてのデバイスを配置したショーケース |
| 通信キャリア網構成 | Level3・MegaPath・拠点FW・PSTN等を含む本格的な構成 |

## ダウンロード

最新版は [Releases ページ](https://github.com/tokotokokame/net-simulation/releases/latest) からダウンロードできます。

### Android

対応OS: Android 8.0（API 26）以上

ファイル名: `net-simulation-android-vX.X.X.apk`

インストール手順:
1. APKファイルをダウンロードする
2. 設定 → セキュリティ → 提供元不明のアプリ を許可する
3. APKをタップしてインストールする

### iOS（ベータ版）

対応OS: iOS 13.0 以上

ファイル名: `net-simulation-ios-vX.X.X.ipa`

本ビルドは署名なし版です。実機へのインストールには以下のいずれかが必要です。

- Apple Configurator 2 または Xcode を使った手動インストール
- Apple Developer Program への登録（年間99ドル）経由のTestFlight配布

動作確認はiOSシミュレーターで行っています。

## ローカルビルド

```bash
# リポジトリをクローン
git clone https://github.com/tokotokokame/net-simulation.git
cd net-simulation

# 依存パッケージを取得
flutter pub get

# テストを実行
flutter test

# Android向けビルド
flutter build apk --release

# iOS向けビルド（macOS + Xcodeが必要）
cd ios && pod install && cd ..
flutter build ios --release --no-codesign
```
