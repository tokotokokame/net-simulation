# Net.Simulation

プロフェッショナル仕様のモバイルネットワークシミュレータ

## ダウンロード

**👉 [最新リリース](https://github.com/tokotokokame/net-simulation/releases/latest)**

### 🤖 Android

| 項目 | 内容 |
|------|------|
| ファイル | `net-simulation-android-vX.X.X.apk` |
| 対応OS | Android 8.0 (API 26) 以上 |

**インストール手順:**
1. `net-simulation-android-*.apk` をダウンロード
2. 設定 → セキュリティ → 提供元不明のアプリ を許可
3. APK をタップしてインストール

---

### 🍎 iOS

| 項目 | 内容 |
|------|------|
| ファイル | `net-simulation-ios-vX.X.X.ipa` |
| 対応OS | iOS 13.0 以上 |
| 備考 | 署名なし版。実機インストールには Apple Developer アカウントが必要 |

**インストール手順（TestFlight経由）:**
1. `net-simulation-ios-*.ipa` をダウンロード
2. Apple Configurator 2 または Xcode で実機にインストール
3. または TestFlight でベータ配布（Apple Developer アカウント必要）

> **iOS は現在ベータ版です。** 動作確認はシミュレーターで行っています。
> 実機インストールには Apple Developer Program への登録（年間$99）が必要です。
